## Filing Rules

- File a query answer here when it combines multiple existing pages, is likely to be asked again, and does not naturally belong in `wiki/comparisons/`, `wiki/concepts/`, or `wiki/entities/`.
- Keep the answer claim-typed. Sourced claims still use `[!source]`; cross-page synthesis still uses `[!analysis]`.
- Add the answer page to this table and append a `query` entry to [[log]].
- Rebuild [[index]] only when the filed answer is a knowledge node. Most files in this folder should be `type: meta` and stay out of the generated index.
